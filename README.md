# 🔐 Password-Based Smart Door Lock System using Arduino

## 📌 Project Overview
This project is a smart security door lock system developed using **Arduino UNO**.  
A user enters the password through a **4x4 keypad**.  

- If the password is correct, the **servo motor unlocks the door**.  
- If the password is wrong, the **buzzer sounds** and **red LED turns ON**.  
- LCD displays messages such as password status.

This project is useful for **home security, locker systems, and embedded system learning**.

## 🔌 Pin Connections

| Component | Arduino Pin |
|----------|-------------|
| Servo Motor | D10 |
| Buzzer | D11 |
| Green LED | D12 |
| Red LED | D13 |
| LCD | D2 - D7 |
| Keypad | D8, D9, A0 - A5 |

---

## 🧠 Working Principle

1. User enters password using keypad.  
2. Arduino checks entered password.  
3. If password is correct:  
   - Servo rotates and unlocks door  
   - Green LED ON  
4. If password is wrong:  
   - Red LED ON  
   - Buzzer sounds  
5. LCD displays status messages.
